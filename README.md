# tssocks5
SOCKS v5 protocol implement by Typescript

### Build
```
$ npm build
```

# Features
- [x] Socks5 server
- [x] Support IPv6
- [x] Support Connect command
- [x] Support DNS query cache
- [ ] Support Username/Password authentication
- [ ] Support Bind command
- [ ] Support Udp Associate
- [ ] Socks5 client
